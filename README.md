# MCP Server AntV

为 AntV 可视化库提供文档查询和智能问答的 MCP (Model Context Protocol) 服务器。

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 构建项目

```bash
npm run build
```

### 运行服务器

```bash
npm start
```

## MCP Inspector 调试

使用官方 MCP Inspector 工具进行可视化调试：

```bash
npx @modelcontextprotocol/inspector dist/index.js
```
